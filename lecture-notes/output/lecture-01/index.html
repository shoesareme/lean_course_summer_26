A finite, checkable justification

Mathematics needs somewhere to start:

  • basic objects and definitions;

  • axioms accepted as starting points;

  • rules that produce new statements.

Correct means every step is licensed

A formal proof is a sequence of statements in which every line is:

an axiom,

an assumption currently in scope, or

the result of applying a valid rule to earlier lines.

One rule already carries us far

From P \to Q and P, infer Q.

example (P Q : Prop) (implication : P → Q) (proofOfP : P) : Q := implication proofOfP

Lean checks that the proof term has exactly the type claimed by the theorem.

Contradictions have no proof

P \land \neg P says that P and “not P” both hold.

example (P : Prop) : ¬ (P ∧ ¬ P) := byP:Prop⊢ ¬(P ∧ ¬P) intro hP:Proph:P ∧ ¬P⊢ False exact h.2 h.1All goals completed! 🐙

Assuming the contradiction gives both h.1 : P and h.2 : P → False.

Lean separates the claim from its evidence

The proposition

example (n : Nat) : Prop := n + 0 = n

Its proof

example (n : Nat) : n + 0 = n := byn:Nat⊢ n + 0 = n simpAll goals completed! 🐙

The kernel accepts the theorem only after checking the evidence.

Quantifiers turn formulas into statements

#check∀ (n : Nat), n + 0 = n : Prop ∀ n : Nat, n + 0 = n
∀ (n : Nat), n + 0 = n : Prop
#check∃ n, n + 1 = 5 : Prop ∃ n : Nat, n + 1 = 5
∃ n, n + 1 = 5 : Prop
#check∀ (x y : Nat), x + y = y + x : Prop ∀ x y : Nat, x + y = y + x
∀ (x y : Nat), x + y = y + x : Prop

A variable alone is not a complete claim. A quantifier says which values the claim covers.

Natural numbers begin with zero and successor

The notes describe natural numbers from a logical perspective:

inductive CourseNat where | zero : CourseNat | succ : CourseNat → CourseNat

  • zero is a natural number;

  • succ n is the successor of n;

  • every CourseNat is built from these constructors.

Addition is determined recursively

def CourseNat.add : CourseNat → CourseNat → CourseNat | x, .zero => x | x, .succ y => .succ (add x y)

These two equations are the computational form of the addition axioms:

x + 0 = x

x + S(y) = S(x + y)

Some axioms become computation

Because addition was defined by recursion on its second input, Lean proves both equations by reducing the definition.

theorem CourseNat.add_zero (x : CourseNat) : CourseNat.add x .zero = x := byx:CourseNat⊢ x.add zero = x rflAll goals completed! 🐙 theorem CourseNat.add_succ (x y : CourseNat) : CourseNat.add x (.succ y) = .succ (CourseNat.add x y) := byx:CourseNaty:CourseNat⊢ x.add y.succ = (x.add y).succ rflAll goals completed! 🐙

Induction handles every natural number

To prove \varphi(n) for all natural numbers:

Base case

Prove \varphi(0).

Inductive step

Assume \varphi(n) and prove \varphi(S(n)).

The two cases match the two constructors of CourseNat.

Induction proves zero is a left identity

theorem CourseNat.zero_add (n : CourseNat) : CourseNat.add .zero n = n := byn:CourseNat⊢ zero.add n = n induction n with | zero =>⊢ zero.add zero = zero rflAll goals completed! 🐙 | succ n ih =>n:CourseNatih:zero.add n = n⊢ zero.add n.succ = n.succ simp [CourseNat.add, ih]All goals completed! 🐙

The definition computes on the second argument, so this direction needs induction.

The first substantial target is commutativity

The lecture notes end with:

\forall x\,y,\; x + y = y + x

In Lean:

theorem CourseNat.add_comm (x y : CourseNat) : CourseNat.add x y = CourseNat.add y x := byx:CourseNaty:CourseNat⊢ x.add y = y.add x induction y with | zero =>x:CourseNat⊢ x.add zero = zero.add x exact (CourseNat.zero_add x).symmAll goals completed! 🐙 | succ y ih =>x:CourseNaty:CourseNatih:x.add y = y.add x⊢ x.add y.succ = y.succ.add x calc CourseNat.add x (.succ y) = .succ (CourseNat.add x y) := rfl _ = .succ (CourseNat.add y x) := congrArg CourseNat.succ ih _ = CourseNat.add (.succ y) x := (CourseNat.succ_add y x).symm

What to carry into the course

  1. A statement is precise enough to have a type.

  2. A proof is evidence checked against that type.

  3. Definitions encode our mathematical starting points.

  4. Deduction and induction build new truths from them.
